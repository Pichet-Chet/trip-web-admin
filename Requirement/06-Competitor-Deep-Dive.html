2 Competitor Analysis — Detail

Category A: B2B Tour Operator Suite

MYT SaaS (Manage Your Trip)

ตลาด: Tour Operators & DMCs ทั่วโลก — ราคา: $100+/เดือน (คาดการณ์)
Itinerary Creator CRM Guide App แพง ซับซ้อน ไม่มี LINE
⚔️ MYT = "ระบบหลังบ้านครบ" vs NatGan = "แจ้งเตือนลูกทริปง่ายๆ"
Enterprise-grade ไม่ใช่คู่แข่งตรง — บริษัททัวร์เล็กในไทยไม่ต้องการ CRM ขนาดนั้น

TourSys (ไทย)

ตลาด: Travel agency ไทย
Revenue analysis Sales tracking ไม่เน้น communication Thai market
⚔️ TourSys = "ระบบจัดการธุรกิจทัวร์" vs NatGan = "ระบบสื่อสารกับลูกทริป"
Back-office tool — ไม่มีฟีเจอร์ส่ง itinerary ให้ลูกทริป + แจ้งเตือน

OTRAMS

ตลาด: Global tour operators — Enterprise pricing
Dynamic packaging Multi-supplier Enterprise-grade แพง ไม่เน้น itinerary comm
⚔️ ตลาดคนละ segment — OTRAMS เน้น booking, NatGan เน้น communication

Category B: B2C Trip Planner

Wanderlog

ตลาด: B2C นักเที่ยวทั่วโลก — ฟรี (Pro $5/เดือน)
AI trip planning Collaboration Budget tracking Text-heavy ไม่มี change noti ไม่มี read receipt
⚔️ Wanderlog = "วางแผนร่วมกัน" vs NatGan = "ส่ง plan ให้ดู + แจ้งเมื่อเปลี่ยน"
Trip planner ไม่ใช่ trip communicator — ไม่ตอบโจทย์ "แก้แล้วลูกทริปไม่รู้"

TripIt

ตลาด: B2C นักเดินทางบ่อย — ฟรี (Pro $49/ปี)
Auto-create itinerary Flight alerts Individual only ไม่มี B2B Noti = flight เท่านั้น
⚔️ TripIt = "จัดการทริปตัวเอง" vs NatGan = "บริษัทส่ง plan ให้ลูกทริป"

TripNiceDay (ไทย)

ตลาด: B2C คนไทยวางแผนเที่ยว — ฟรี
เป็นไทย ใช้ง่าย Community B2C only ไม่มี notification
⚔️ TripNiceDay = "วางแผนเที่ยวเอง" vs NatGan = "บริษัทส่ง plan ให้ลูกทริปดู"

Category C: White-label Travel App

mTrip

ตลาด: Travel agencies, TMCs — $500+/เดือน (คาดการณ์)
White-label app Push noti Traveler tracking แพงมาก ต้อง download app Setup ซับซ้อน
⚔️ mTrip = "เหมือน NatGan แต่เป็น app แยก ราคาแพง" — NatGan = web-based ไม่ต้องลง app
⚡ ใกล้เคียง NatGan ที่สุด แต่เป็น enterprise solution — SME ไทยจ่ายไม่ได้

AXUS Travel App

ตลาด: Luxury travel advisors — $35-150/เดือน
Change notification! In-app messaging Beautiful UI Luxury market English only ต้อง download app
⚔️ AXUS มี change noti เหมือน NatGan แต่ต้อง download app + แพง + ไม่มี LINE

Category D: DIY — คู่แข่งตัวจริง

LINE Group Chat — ส่วนแบ่ง ~90%

บริษัททัวร์ไทย ~90% ใช้อยู่
ฟรี ทุกคนมี LINE ส่งได้ทันที ข้อมูลจมในแชท หาข้อมูลไม่เจอ ไม่มี read receipt สำหรับ itinerary
⚔️ LINE = "ส่งข้อความ" vs NatGan = "ส่ง itinerary ที่ always up-to-date + รู้ว่าใครเห็นแล้ว"

PDF / Canva — ส่วนแบ่ง ~40%

ใช้ร่วมกับ LINE
สวย ปริ้นได้ Professional Static — แก้ทีต้องทำใหม่ ลูกทริปอาจดูไฟล์เก่า
⚔️ PDF = snapshot ณ วันที่ทำ vs NatGan = live link ที่อัพเดทตลอด

ความเสี่ยงจากคู่แข่ง

ต่ำ

LINE เพิ่มฟีเจอร์ itinerary ใน LINE OA

LINE ไม่เคยทำ vertical solution — ถ้าเกิดขึ้น เรามี read receipt + immigration view ที่ LINE ไม่มี

กลาง

Wanderlog เพิ่ม B2B mode

เป็นไปได้ แต่ Wanderlog ไม่มี LINE integration + ไม่มีภาษาไทย

กลาง-สูง

คนไทยทำ clone NatGan

ถ้าเราสำเร็จ → First-mover advantage + viral loop + community lock-in

สูง

บริษัททัวร์ใช้ LINE ต่อเพราะ "ก็ใช้ได้อยู่"

Inertia แรงมาก — ต้องทำให้ NatGan ดีกว่า LINE อย่างเห็นได้ชัด (read receipt = killer feature)

NatGan ไม่มีคู่แข่งตรงในตลาดไทย

B2B Suite → แพงเกินไป   |   Trip Planner → ไม่ใช่ B2B   |   White-label → ต้อง download app

คู่แข่งที่แท้จริง = "ความเคยชิน" ของการใช้ LINE Group

ไม่ต้องทำให้ดีกว่า LINE ทุกด้าน — แค่ทำให้
"รู้ว่าใครเห็นแล้ว" ดีกว่า LINE ก็พอ
